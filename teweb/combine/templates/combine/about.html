About

Tellurium Web Tools provide a simple means for the execution of CombineArchives.

The tellurium Web Tools allow to execute CombineArchives with model definition in SBML and simulation definitions in SED-ML.

For more information about the Simulation Experiment Markup Language see the SED-ML Website. For more information about the Systems Biology Markup Language see the SBML-Webseite

version: {{ commit }}

Features

Currently, CombineArchives with model descriptions in SBML and simulation descriptions in SED-ML are supported.
